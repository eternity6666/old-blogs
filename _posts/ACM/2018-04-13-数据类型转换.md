## 问题描述

如何将C++里的int类型转换为string类型

## 解决方案

使用<sstream>库中的stringstream流

## CODE
```
#include <iostream>
#include <sstream>
using namespace std;

int main()
{
    int a=1000;
    string str;

    stringstream ss;
    ss<<a;
    ss>>str;

    cout<<a<<endl;
    cout<<str<<endl;

    return 0;
}
```

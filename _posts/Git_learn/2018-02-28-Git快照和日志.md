查看git提交纪录
```
git log
```

```git log```参数说明
```
git log -1//获得最近一次提交的纪录
```
```
git log -2//获得最近两次提交的纪录
```
```
git log --oneline//一次提交纪录一行输出
```
```
git log -1 -p//获得最近一次提交的纪录和修改内容情况
```

使用图形化的方式查看提交纪录:```gitk```

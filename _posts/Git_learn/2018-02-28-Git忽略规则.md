要让Git忽略某些文件或文件夹，只需在.gitignore文件中添加

**忽略某个文件(如忽略a.o文件)**
```
echo "a.o" >> .gitignore
```
**忽略某类文件(如忽略a.o和b.o等)**
```
ehco "*.o" >> .gitignore
```
**忽略某个文件夹(如忽略code文件夹)**
```
echo "code/" >> .gitignore
```
